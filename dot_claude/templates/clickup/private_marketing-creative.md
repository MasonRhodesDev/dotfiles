# ClickUp Template: Marketing/Creative Request

**Use for:** Email campaigns, creative assets, marketing tasks, brand-specific work
**Target length:** 200-800 characters
**Tone:** Informal, conversational, brief

---

## Template

```markdown
BRAND - CATEGORY - TYPE - Brief Description

[Brief description of request - often pasted from Slack or form]

[Optional: Include verbatim Slack messages or original request]

Enter Your Email:
[requester email]

Brand:
[Brand name]

Category:
[Marketing/Martech/Operations/Creative/etc]
```

---

## Alternative Template (More Structured)

```markdown
BRAND - CATEGORY - TYPE - Brief Description

## Request

[What needs to be created or done]

## Context

[Any background information, links to examples, related campaigns]

## Deliverables

- Deliverable 1
- Deliverable 2
- Deliverable 3

Enter Your Email: [requester]
Brand: [brand]
Category: [category]
```

---

## Filling Instructions

**Title Format:**
- `BRAND - CATEGORY - TYPE - Description`
- Examples:
  - `LFMD - WMP - Email - Testimonial Series`
  - `RMD - Valentine's Day - Email Designs w/ $100 Credit Promo`
  - `SMDO - Social - Instagram Posts - Q2 Campaign`

**Description:**
- Keep conversational and brief
- Paste original Slack messages if request came from Slack
- Include @mentions as they appeared originally
- Don't over-format or make too formal
- Links to examples, mockups, previous work are helpful

**Enter Your Email:**
- Requester's email for tracking
- Use custom field if available

**Brand:**
- LifeMD, Rex MD, Shapiro MD, Nava MD, etc.
- Use custom field if available

**Category:**
- Marketing, Martech, Creative, Operations, Social, Email, etc.
- Use custom field if available

## Metadata to Set

**Tags:** Brand tag, campaign tag if applicable (usually 1-2 tags)
**Priority:** Normal unless time-sensitive campaign
**Story Points:** Usually 1-3 for creative work
**List:** Marketing list or brand-specific list
**Assignee:** Marketing team member or designer
**Custom Fields:**
- Brand (required)
- Category (required)
- Requested By (optional)

---

## Tone Guidelines

**DO:**
- Keep it conversational and informal
- Paste Slack messages verbatim
- Include @mentions as they appeared
- Use casual abbreviations: "w/" for "with", "promo" for "promotion"
- Link to examples and references liberally
- Keep formatting minimal

**DON'T:**
- Over-formalize the language
- Write long explanatory paragraphs
- Use technical jargon
- Create detailed specifications (unless requested)
- Remove personality from original request

---

## Length Guidelines

**200-400 chars:** Minimum for simple requests
**400-600 chars:** Good for most marketing tasks
**600-800 chars:** Maximum before too detailed

Marketing requests tend to be brief and direct. If you're writing more than 800 chars, consider if you're over-specifying.

---

## When to Use This Template

Use marketing-creative template when:
- Request is for marketing, creative, or brand work
- Originated from marketing team or Slack
- Focus is on deliverable type (email, social post, ad, etc.)
- Informal, conversational tone is appropriate
- Brand-specific work

Use complex-feature template when:
- Technical implementation required (Martech platform config)
- Multiple systems integration
- Needs developer involvement

---

## Common Sections to Include

**If email campaign:**
- Number of emails in series
- Target audience
- Send dates/timeline
- Subject line ideas (optional)
- Reference to past campaigns

**If social media:**
- Number of posts
- Platform(s)
- Content themes
- Hashtag strategy (if applicable)
- Visual style references

**If creative assets:**
- Dimensions/formats needed
- Use case (web, print, social, email)
- Brand guidelines reference
- Deadline
